## Shortcuts

* `ctrl` + `\` shows/hides the Tree View.  
* `ctrl` + `j` adds the next line to the current.  
* `ctrl` + `shift` + `d` duplicates the current line.  
* `ctrl` + `shift` + `k` deleted the current line.
